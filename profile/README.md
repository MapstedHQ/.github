Welcome to the official GitHub home of **Mapsted** — the public distribution point for the **Mapsted Mobile SDK** and its sample projects. Our technology delivers accurate indoor positioning and turn‑by‑turn wayfinding **without beacons, Wi‑Fi triangulation, or any on‑site hardware**.

## 📦 Mobile SDK

The SDK ships as modular components. Integrate only what you need.

| Area | Repositories |
|------|--------------|
| **Positioning** | [`mapsted-sdk-positioning`](https://github.com/MapstedHQ/mapsted-sdk-positioning) · [`mapsted-sdk-core-spm`](https://github.com/MapstedHQ/mapsted-sdk-core-spm) |
| **Maps & Wayfinding** | [`mapsted-sdk-map`](https://github.com/MapstedHQ/mapsted-sdk-map) · [`mapsted-sdk-map-ui`](https://github.com/MapstedHQ/mapsted-sdk-map-ui) · [`core-map-sdk`](https://github.com/MapstedHQ/core-map-sdk) |
| **Location Marketing** | [`mapsted-sdk-loc-marketing`](https://github.com/MapstedHQ/mapsted-sdk-loc-marketing) |
| **Geofencing** | [`mapsted-sdk-geofence`](https://github.com/MapstedHQ/mapsted-sdk-geofence) |
| **CocoaPods specs** | [`podspec`](https://github.com/MapstedHQ/podspec) |
| **Swift Package Manager** | [`mapsted-sdk-map-spm`](https://github.com/MapstedHQ/mapsted-sdk-map-spm) · [`mapsted-sdk-core-map-spm`](https://github.com/MapstedHQ/mapsted-sdk-core-map-spm) |
